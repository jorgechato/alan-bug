# Alan el preguntador
## Install
```zsh
$ git clone https://github.com/orggue/alan-el-preguntador.git
$ cd alan-el-preguntador
$ npm install
$ bower install
```
### Dependencies
You need to install mongodb, nodejs and gulp.
## Run
```zsh
$ gulp # developer mode
$ gulp serve # developer mode
$ gulp compile # prepare enviroment to deploy
```
